# Configurer le projet

Pour lancer le projet suivez les commandes suivantes:
- Cloner le projet:
  ```bash
  git clone https://github.com/TimeoBlondeleauDubois/GestionFilm.git
- Rentrer dans le dossier racine:
  ```bash
  cd GestionFilm
- Récupérer les packages:
  ```bash
  npm install

- Veuillez créer votre .env en mettant les mêmes variables que dans le .env.template. Vous pouvez obtenir votre clé api en allant sur le site [MovieDb](https://www.themoviedb.org/)  puis une fois connecté cliquez sur accédez à votre compte, puis dans Settings > API, générez une clé API.

- Pour finir lancer le projet avec:
  ```bash
  npm run dev
